# macOS-foldericon
Icons of folders in MacOS.

# Build icns file
png2icns.sh is shell script to build icns file ans iconset dolder.

png2icns.sh needs ImageMagick and iconutil.

Execute a following command in a terminal.
```bash
./png2icns.sh [image path]
```
